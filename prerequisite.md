# 基础技能培训第2讲课前准备



## 请安装以下软件或工具，遇到问题请在群里提出：

1. VS Code      请参考[Doc9](https://docs.net9.org/basic/vscode/)。
2. g++ 或 clang
3. gdb 或 lldb
4. make

check point: 能在命令行中执行以下命令

1. `code -v`, 
2. `g++ -v`（或使用clang）
3. `gdb -v` (或lldb)
4. `make -v`

## 可选，可能有助于上课体验：

typora 或其他 markdown 编辑器

## 涉及，但不与课程内容直接相关：

pandoc, xelatex
